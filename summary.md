## API 入門教學總覽

以下為本次課程的五個獨立教學文件，涵蓋從 API 基礎概念到實作、安裝與安全性、以及 RESTful 設計原則。

| # | 文件 | 主要內容 |
|---|------|----------|
| 1 | `API_Intro_01_WhatIsAPI.md` | API 定義、餐廳比喻、技術定義、參數傳遞技術層面 |
| 2 | `API_Intro_02_HTTPMethods.md` | HTTP 方法與 CRUD 對應、實戰範例、參數傳遞類型與差異 |
| 3 | `API_Intro_03_DotNetImplementation.md` | ASP.NET Core CRUD 範例、路由、狀態碼、`CreatedAtAction`、參數綁定說明 |
| 4 | `API_Intro_04_Security.md` | 安全性議題（身份驗證、授權、CORS、輸入驗證） |
| 5 | `API_Intro_05_RESTful.md` | RESTful 原則、資源與 HTTP 方法、URI 設計、最佳實踐 |
