# API 入門總結概要

這份文件概括了 API 開發與使用的核心概念，重點覆蓋以下主題：

1. **API 定義與功能** – 說明 API（Application Programming Interface）的意義以及它將資料存取、操作與系統間協作抽象化的作用。
2. **HTTP 方法** – 詳列 GET、POST、PUT、PATCH、DELETE 等常用方法，並提供實際範例，說明每種方法的用途。
3. **.NET 實作** – 節錄的 ASP.NET Core Web API 範例，展示 Controller、路由和返回 JSON 的基本寫法。
4. **安全性** – 強調常見風險（SQL 注入、XSS、CSRF）及對應措施（JWT/OAuth、HTTPS、輸入驗證）。
5. **RESTful 設計原則** – 包括資源化、無狀態、標準 HTTP 方法、URL 層級、HATEOAS 等最佳實踐。
6. **cURL 範例** – 展示使用 cURL 進行 CRUD 操作及帶認證請求的命令。
7. **工具介紹** – 介紹 Postman、Swagger/OpenAPI、Insomnia、cURL、JMeter、Charles Proxy 等 API 開發、測試與監控工具。

整體而言，文件以清晰的章節與表格，提供初學者快速掌握 API 基礎、實務範例以及安全與測試工具的使用說明，為後續進一步的 API 開發奠定基礎。
